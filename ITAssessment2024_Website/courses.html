Computer Science

Description of Computer Science:

This is a general introduction to the principles and practices of computer science. Topics include algorithms, data structures, and more.

  • Programming and Algorithms: Students learn various programming languages and study algorithms and data structures.
  • Software Development: Courses cover software engineering principles, design, testing, debugging, and project management.
  • Database Management: Students learn about database design, SQL, data modeling, and management systems.
  • Computer Networks: Topics include network protocols, security, administration, and communication technologies.
  • Operating Systems: Students study OS concepts, processes, memory management, file systems, and virtualization.

Duration: 4 Years

Course Details

  • The course is divided into eight semesters.
  • There are two exams in each semester - one midway and another at the end of the semester.
  • A project will be assigned to every student, which they have to submit by the last date of the examination period.
  • A total of 6 credits will be awarded if all subjects are passed.
